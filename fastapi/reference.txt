* 파이썬 가상환경 생성 방법
참고 - Ubuntu linux 환경에서 파이썬 가상환경 생성 하려면 "sudo apt install python3.10-venv" 터미널 명령어 먼저 실행 필수
참고: https://claude.ai/chat/9ad65b04-2a86-456e-b655-cd8f3d1fae8c

비쥬얼스튜디오코드 (VSCode) cmd 터미널창 열고 -> 터미널 명령어 python -m venv "가상환경이름" 입력 및 엔터 
예) Window - python -m venv .fastapi_env(가상환경 폴더 숨김 처리) / Ubuntu linux - python3 -m venv fastapi_env
가상환경 폴더 ".fastapi_env" 안에는 앞으로 가상환경에 설치할 모든 라이브러리 소스코드가 저장된다. (예) 폴더 Include, Lib, Scripts 등등...
(예) python -m venv .codeprj1 (.codeprj1 - 가상환경 폴더 숨김 처리)

* 파이썬 가상환경 활성화 방법
비쥬얼스튜디오코드 (VSCode) cmd 터미널창 열고 -> 터미널 명령어 "가상환경이름"\Scripts\activate.bat 입력 및 엔터
예) Window - .fastapi_env\Scripts\activate.bat / Ubuntu linux - source fastapi_env/bin/activate
위의 명령어가 뜻하는 것은 새로 생성한 가상환경 ".fastapi_env" 안에 폴더 Scripts 안에 activate.bat 파일을 실행한다는 뜻이다.

가상환경이 활성화 되면 아래처럼 cmd 터미널창 맨 왼쪽에 (가상환경이름)이 생성된다.
(예) (fastapi_env) D:\minjae\test_FastAPI\fastapi>

가상환경이 활성화 되면 이제 cmd 터미널창 안에서 설치하는 모든 라이브러리는 해당 가상 환경 안에 설치 된다.

* 파이썬 스크립트 파일 (lambda_function.py)과 새로 설치한 가상환경 (fastapi_env) 연결 방법 
1. 비쥬얼스튜디오코드 (VSCode) 상단 리본 탭 "View" 클릭 -> 컨텍스트 메뉴 "Command Palette..." 클릭 
2. Python: Select Interpreter 검색 및 클릭
3. 방금 전에 설치했던 가상환경 "Python 3.11.4 ('fastapi_env') .\fastapi_env\Scripts\python.exe" 클릭
4. 해당 파이썬 스크립트 파일 (lambda_function.py)은 방금 전 생성한 가상환경 "fastapi_env"와 연결 되어 해당 가상환경 안에서 작업 가능하다.

* 파이썬 스크립트 파일 (lambda_function.py) 실행 방법
비쥬얼스튜디오코드 (VSCode) cmd 터미널창 열고 -> 터미널 명령어 python lambda_function.py 입력 및 엔터
